# Roadmap

## Foundation

Apache-2.0 monorepo, Go core, Python analyzers, NATS JetStream, PostgreSQL, CLI, docs, threat model, Docker Compose.

## Artifact Observer

npm/PyPI metadata observation, artifact download, local hash validation, same-version mutation detection, safe unpacking, snapshots, diffs.

## Analyzer Engine

Versioned analyzer contracts, Python analyzer runtime, deterministic rules, finding persistence, findings API.

## Dependency Graph + SCM

Direct/transitive graph, compromise propagation, GitHub adapter, source correlation. Roadmap source providers: GitLab, Codeberg, Forgejo/Gitea, Bitbucket, generic Git/local repo.

## Alerts + UI

Auth abstraction with local auth and GitHub OAuth. Email and Microsoft Teams notifications. React triage UI. REST/OpenAPI.

## Expansion

Go modules and Rust crates next. Later Maven, NuGet, RubyGems, Packagist, OCI/Docker. Optional LLM narrative enrichment. Optional opt-in community signal exchange.
